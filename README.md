# 鮮食空投箱 FreshDrop

## 網站介紹：
鮮食空投箱是一個虛擬的鮮食訂購平台。使用者可以選購菜色，然後在家收到簡單快速料理的鮮食食材包，以解決現代人忙碌沒時間烹煮和購買食材的問題。<br>

## 使用技術：
* Javascript ES6 以上標準
* Vite
* Scss 進行設計系統規範
* D3.js 製作圖表與台灣地圖互動效果
* RWD 響應式網頁設計
* Git 版本控制

## 網站架構：
* 首頁
* 候選人介紹頁
* 歷屆票選結果
* 開票倒數
