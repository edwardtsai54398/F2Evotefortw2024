# 鮮食空投箱 FreshDrop

## 網站介紹：
虛擬的貓咪總統大選全台開票網－圖像化得票數據，簡單易懂。讓使用者輕易掌握最新選情動態!<br>
<!-- [展示頁面 Display page]() -->

## 使用技術：
* Javascript ES6 以上標準
* Vite
* Scss 進行設計系統規範
* D3.js 製作圖表與台灣地圖互動效果
* RWD 響應式網頁設計
* Git 版本控制

## 網站架構：
* 首頁
* 候選人介紹頁
* 歷屆票選結果
* 開票倒數
